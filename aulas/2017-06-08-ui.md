## CLI: Interface de Linha de Comando
* No Ubuntu:
  * CTRL+ALT+T
    ```echo "Fulano de Tal"
    ls
    ls -1
    cd Documentos
    pwd
    echo {1..10}
    echo {1..50}
    mkdir casa
    mkdir comida
    mkdir roupa lavada
    mkdir "roupa lavada"
    mkdir -v pasta-{01..30}
    ```
  * Abra o **Gerenciador de arquivos* e verifique a pasta **Documentos**
  
    
* No Windows
  * \<Windows\> + E: `Explorer` (Gerenciador de arquivos)
  * \<Windows\> + F: `Find` (Pesquisar)
  * \<Windows\> + D: `Desktop` (Exibir a Área de Trabalho)
  * \<Windows\> + R: `Run` (Executar)
  * Prompt de comandos:
    * Antigo:   
      * \<Windows\> + R: `cmd`
    * Mais novo: 
      * \<Windows\>: Power Shell
      
   * Comandos do DOS:
      * \<Windows\> + R: `cmd`
      ```dir
      cd Desktop
      mkdir casa
      mkdir comida
      mkdir roupa lavada
      mkdir "roupa lavada"
      dir```
